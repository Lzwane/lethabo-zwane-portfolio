# Lethabo Zwane Portfolio

This is the professional portfolio website for Lethabo Zwane, showcasing projects, certifications, and professional experience as a Deputy CTO and Software Engineer.

## Project Tech Stack

This project is built with:

- **Vite**: For fast development and building.
- **React**: UI library.
- **TypeScript**: For type-safe code.
- **shadcn/ui**: For accessible and customizable UI components.
- **Tailwind CSS**: For styling.
